First steps with Jekyll!

Mar 29, 2020

Never used Ruby at all but after trying Pelican and realising that I’d found no need for any Python I thought I’d give Jekyll a go.

I like the idea of a static site generator for several reasons - security, simplicity and the potential to tie in with version control. Security is the biggest of these, using Wordpress for several years, there’s often high profile issues. I’ve also found Wordpress to be getting a bit bloated as it matures and the new block editor is terrible.

In tandem with this is a burdgening adoration for Markdown, which both Pelican and Jekyll support.

Following the step by step tutorial took less than an hour, which is about as much as of attention span as I have now and it’s a credit to the writer that I only drifted off on the last page.
